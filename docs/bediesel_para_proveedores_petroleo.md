# Bediesel para Proveedores de Petróleo

Versión inicial – Propuesta de partnership

---

## 1. Idea general

**Quiénes:**
- Empresas proveedoras de diésel para constructoras, mineras mediana escala, forestales, agrícolas, etc.
- Que venden combustible a clientes que operan con camiones surtidores propios, estanques móviles/fijos o abastecimiento en faena.

**Qué propones:**
- Ofrecer **Bediesel** como un servicio adicional/bundle junto con el suministro de combustible, para que sus clientes controlen mejor el consumo y el rendimiento de sus máquinas.

**Beneficio para el proveedor:**
- Diferenciarse de otros proveedores de diésel que compiten solo por precio.
- Reducir reclamos y discusiones por diferencias en consumos.
- Fidelizar clientes (el cliente se “amarra” al proveedor que le ofrece control y visibilidad).

**Beneficio para el cliente final:**
- Controla cada litro que recibe del proveedor.
- Ve rendimientos por máquina y proyecto.
- Reduce pérdidas y tiempo en cuadrar combustibles.

## 2. Propuesta de valor específica para el proveedor

### 2.1. Mensaje central

> "No solo te vendo diésel.  
> Te ayudo a que tus clientes controlen cada litro y se queden contigo porque confían en sus números."

### 2.2. Beneficios clave

1. **Diferenciación comercial**
   - Mientras otros compiten por centavos en el precio del litro, el proveedor con Bediesel ofrece:
     - Plataforma de control de consumos.
     - Reportes de rendimientos por máquina y obra.
   - Esto se puede presentar como: "Plan combustible + control".

2. **Relación de confianza**
   - Si el cliente tiene dudas de pérdidas o diferencias, el proveedor puede decir:  
     "Controlemos juntos cada carga con Bediesel".
   - Menos discusiones y más datos.

3. **Fidelización y permanencia**
   - Un cliente que monta su control de combustible sobre el ecosistema del proveedor tiene más costos de cambio.
   - Si el proveedor se va, el cliente pierde parte de su solución, lo que desincentiva cambiar de proveedor solo por precio.

4. **Posible nuevo revenue stream**
   - El proveedor puede:  
     - Revender Bediesel con un margen por cliente.  
     - O usarlo como valor agregado incluido para clientes que superan cierto volumen de litros mensuales.

## 3. Modelos de colaboración sugeridos

### 3.1. Revendedor con margen

- Besimplit cobra al proveedor el precio base (ej: 1 UF por cada 10.000 L).  
- El proveedor lo vende al cliente final con un markup (ej: 1,2–1,5 UF por cada 10.000 L, embebido dentro de su "plan de abastecimiento").
- El proveedor factura al cliente final; Besimplit factura al proveedor.

Ventajas:
- El proveedor gana un margen adicional asociado al servicio.
- El cliente ve una única relación comercial para diésel + control.

### 3.2. Valor agregado para clientes clave

- El proveedor absorbe el costo de Bediesel para ciertos clientes estratégicos (por volumen de litros o antigüedad).
- Se presenta como beneficio exclusivo, por ejemplo:  
  "Clientes sobre X litros/mes tienen acceso sin costo a la plataforma de control de combustible".

Ventajas:
- Herramienta de fidelización fuerte sin aumentar el precio aparente del litro.

### 3.3. Co-branding

- En la interfaz de Bediesel se puede mostrar el logo del proveedor junto al de Besimplit.
- En la comunicación con el cliente final se habla de:  
  "[Nombre Proveedor] + Besimplit: Control de combustible para tus obras".

Ventajas:
- Refuerza la marca del proveedor como "tecnológico" y no solo "logístico".

## 4. Argumentario comercial para hablar con proveedores

### 4.1. Preguntas para abrir la conversación

> 1. ¿Qué tan seguido tus clientes se quejan de que el combustible "no cuadra" o que el consumo parece más alto de lo esperado?  
> 2. Cuando eso pasa, ¿qué herramientas tienen para demostrar que estás entregando los litros correctos?  
> 3. ¿Tienes hoy alguna solución para ayudar a tus clientes a controlar el rendimiento de sus máquinas, o solo les puedes entregar guías y facturas?

### 4.2. Pitch corto

> Hoy la mayoría de los proveedores de diésel compite solo por precio, pero tus clientes tienen otro problema: **no saben con claridad dónde se va el combustible**.  
> Eso genera sospechas, reclamos, y conversaciones incómodas contigo.
>
> Lo que hacemos en Besimplit con **Bediesel** es registrar cada carga desde el camión surtidor o estanques hacia las máquinas, directamente en una app.  
> Así, tus clientes pueden ver rendimientos por máquina y proyecto, y tú puedes decir:  
> "Acá tienes una herramienta para controlar cada litro que te entrego".
>
> Eso te diferencia de otros proveedores y te ayuda a fidelizar a los clientes buenos.

### 4.3. Cierre hacia demo conjunta

> Te propongo hacer una demo de 20–30 minutos donde te muestro:  
> - cómo funciona el registro desde el camión surtidor,  
> - cómo se ven los reportes por máquina y obra,  
> - y algunos ejemplos de ahorros típicos.
>
> Si te hace sentido, después vemos cómo lo empaquetamos para tus clientes (revenderlo, incluirlo para clientes sobre cierto volumen, etc.).

## 6. Próximos pasos si un proveedor se interesa

1. **Reunión de exploración (discovery)**
   - Entender tamaño del proveedor (litros/mes, tipos de clientes).
   - Ver si atiende constructoras, mineras, forestales, etc.
   - Identificar clientes objetivo para un piloto.

2. **Definir modelo (revendedor vs valor agregado)**
   - ¿Quiere ganar margen extra? → Modelo revendedor.  
   - ¿Quiere usarlo como beneficio para cuentas clave? → Modelo valor agregado.

3. **Piloto con 1–3 clientes del proveedor**
   - Elegir clientes con camión surtidor/estanques y consumo relevante (>30.000 L/mes).  
   - Implementar Bediesel con acompañamiento de Besimplit.  
   - Medir resultados en 2–3 meses (pérdidas detectadas, tiempo administrativo ahorrado, percepción de confianza).

4. **Escalamiento**
   - Si el piloto funciona, empaquetar la oferta para más clientes del proveedor.

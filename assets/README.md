# Carpeta de assets de Besimplit

Este directorio agrupa los recursos visuales (imágenes) que se usan en landings, presentaciones, Remotion, etc.

## Estructura

- `assets/brand/`
  - `logo-besimplit.png` → Logo principal de Besimplit (color).

- `assets/bediesel/`
  - `dashboard_petroleo.png` → Dashboard de control de petróleo.
  - `dashboard_rendimientos.png` → Vista de rendimientos.
  - `dashboard_report_maquinas.png` → Reporte por máquina.
  - `iphone app store bediesel.png` → Mockup de app Bediesel en iPhone/App Store.

- `assets/activos/`
  - `dashboard_mantencion.png` → Dashboard de mantenciones de activos.

- `assets/faenas/`
  - `dashboard_costos.png` → Dashboard de costos de faenas.
  - `dashboard_costos_2.png` → Otra vista de costos de faenas.

- `assets/bodega/`
  - `screenshot_bodega.png` → Screenshot de módulo de bodega.

- `assets/checklist/`
  - `screenshot_checklist.jpg` → Screenshot de módulo de checklist.

## Uso sugerido

- Remotion: copiar o referenciar estos assets desde `remotion-official/public/` cuando se necesiten en videos.
- Landings: usar rutas relativas desde el build (según framework/plataforma que use tu sitio).
- Docs comerciales: reutilizar estas imágenes en cotizaciones, PDFs y presentaciones.
